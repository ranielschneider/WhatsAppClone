# WhatsApp Clone

A native Android application cloning the core interface and functionalities of WhatsApp, built to demonstrate modern mobile development practices and the latest tools in the Android ecosystem.

## 🛠 Tech Stack & Architecture

* **Language:** Kotlin
* **UI Framework:** Jetpack Compose (Declarative UI with custom components)
* **Build System:** Gradle (Kotlin DSL + Version Catalogs)
* **Architecture:** MVVM (Model-View-ViewModel) for clean separation of concerns and robust UI state management.
